# ChatLookup

A client-side all-in-one chat utility mod!

## Features

**Message filtering** - an easy way to temporarily filter messages. Write anything in the search bar and only the matches will be shown on chat!

**Chat history saving** - the chat history is saved on your hard disk (can be turned off), so even after restarting the game you will still be able to see it!

**Match highlighting** - you can toggle highlighting the matches of your search. The color of the highlighter can be configured in the settings.

**Message copying** - `Ctrl+Left-click` any chat line to copy it to the clipboard. Also works while filtering - you copy exactly the message you see. 

**And many more!** - message stacking, timestamps, mention detector, smooth chat animations...

Localized in English, Spanish, Russian and Polish.

## Configuration

The mod is highly configurable. You can easily turn off the functions you don't want to use.

In order to do so - simply click the settings icon using your mouse after opening the chat.
